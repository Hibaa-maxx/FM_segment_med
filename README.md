# Foundation Model-Based Segmentation Framework

This repository contains the implementation of a foundation-model-driven segmentation framework developed for medical image analysis research.

The project explores lightweight decoder designs built on top of frozen vision transformer backbones, focusing on 2D slice-based segmentation for clinical imaging data.
